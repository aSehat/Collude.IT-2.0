Reducer Files Go Here:

Please append to this when you add an item.

1. Readme.txt - This
2. Actions Folder - Contains all actions for redux life cycle
3. Reduceres Folder - Contains all reducers for redux life cycle
4. Components Folder - Contains all of the folders for the components
5. Img Folder - Contains all project images
6. Utils Folder - Contains all auxillary utility files
7. App.js - React site backbone 
8. index.js - Renders App.js by putting it into root document
9. App.css - Contains global styling
10. store.js - Creates redux site backbone
